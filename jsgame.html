Javascript

2021

Voor het vak programmeren moesten wij spelen met Javascript. De opdracht was het maken van een game met HTML, CSS en Javascript. Dit was de eerste keer dat ik met Javascript werkte en dat vond ik erg lastig. Ondanks dat ik het toch erg lastig vond, vind ik dat ik toch wel wat goeds in elkaar heb gezet. In deze game kan je van ruimte wisselen door op 'zadelkamer' te klikken en als je op de deur zelf klikt gaat de gezondheid van het paard naar beneden, als de gezondheid op 0 staat ligt het paard en als het paard te eten krijgt staat het weer op. Als je over het paard heen gaat krijg je geluid te horen. In de zadelkamer kan je het paard opzadelen.

Ik heb geleerd hoe je met Javascript verschillende problemen kan oplossen, maar dat je er ook leuke spelletjes mee in elkaar kan zetten. Helaas heb ik de healthbar niet 100% kunnen laten werken op de tweede pagina, maar al met al was ik erg trots op het resultaat.
